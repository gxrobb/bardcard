# Getting Started

1. Clone the project
2. Create a .env file
   - Copy the contents of `.env.example` to `.env`
   - go to thecatapi.com and get an api key
   - set the api key `VITE_CAT_API_KEY=your_api_key_here`
3. run `nvm use` if you are using nvm. Otherwise we are using node v22.18.0.
4. run yarn install
5. run yarn dev
